PokéAPI

Esta página utiliza la api PokéAPI para proporcionar diversos datos de Pokémon. Puedes escoger entre echarle un vistazo a cualquier Pokémon existente y su información, sus hábitats y las diferentes generaciones de videojuegos de las que provienen. Esta web ha sido realizada por Isaac Esteve Alonso, estudiando en IFP Hospitalet en el curso de desarrollo de videojuegos y aplicaciones multimedia.

Puedes empezar a realizar tus búsquedas con las opciones del menú superior de esta página.
